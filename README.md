É fundamental compreender sobre: Controlar Direcionar Extrair valor real de modelos de IA de forma estratégica.

# Objetivo: 🏅

**explora técnicas e metodologias para:**

1-  projetar, otimizar e avaliar interações com modelos de linguagem e sistemas de IA generativa.

3- Avaliar a qualidade das respostas geradas.
